## 1. 문제   

[군집의 중심 각도를 추정하는 NN 만들기]

-----train_data.npy-----
* 각 데이터는 100개의 복소수로 구성
* 20000 × 100 array   
* 20000개 데이터, 각 데이터는 100개의 복소수
* 각 데이터는 동일한 위상에 노이즈가 섞인 것
   
-----target_data.npy-----   
* 20000개의 복소수의 위상 값
* 20000 × 1 array
* 각 범위는 0~2π    
   
-----성능 평가지표------   
* 학습 mse, Test Data에 mse   
* Average((target위상 - 추정한 위상)^2)   
* 입력 노드의 개수는 자유롭게 구성, Layer 수도 자유롭게
